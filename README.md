## 工具列表

### HaHa — 视频审核工作台
视频批量审核工具，支持逐帧查看、标注画笔、状态标记、反馈记录、Excel 导入导出。

**版本：** v1.4  
**下载：** [HaHa_v1.4.zip](https://github.com/yanayang87/my-tools/releases/download/haha-v1.4/HaHa_v1.4.zip)

**功能：**
- 多目录管理，批量加载视频，刷新目录
- 逐帧播放、缩放平移、播放速度调节
- 画笔标注 + 截图保存
- 通过/不通过/待定 状态标记
- 文字反馈绑定帧号，支持编辑
- 时间轴标记 + 快捷键跳转（< >）
- 反馈列表排序（新→旧 / 旧→新）
- 左右侧边栏可收起/展开
- 导出 Excel（含截图）/ 导入 Excel 恢复审核状态
- 状态筛选、侧边栏可拖拽调宽
- 表格拖拽关联视频（手动匹配改名视频）
- 完整快捷键系统（20+ 快捷键，可自定义）
- 设置面板（快捷键修改 + 跳跃步长/速度档位调整）

## 使用方式

1. 点击上方下载链接，下载 zip 文件
2. 解压到任意位置
3. 双击 `.exe` 文件运行

无需安装，不写注册表，删除文件夹即为卸载。

---

## 更新方式

重新下载最新版本覆盖即可。后续版本会加入自动更新功能。
